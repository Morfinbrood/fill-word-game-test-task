# fill-word-game-test-task
Start task at 17:25 30.04.18

Реализуйте на языке JavaScript основной геймплей мобильной игры “Филворды” (таблицу слов, процесс их поиска и возможность начать заново после окончания игры). При реализации допускается использование любых сторонних библиотек.         
Ссылка на филворды: https://play.google.com/store/apps/details?id=com.merigotech.fillwords&hl=ru
Размер таблицы слов может изменяться от 3X3 до 10X10. Так как это mobile-first приложение, оно не должно “плыть” на мобильных устройствах с различным разрешением. 
Реализация анимаций выделения ячеек и красивое выделение правильно угаданных слов будет плюсом. Любая обоснованная креативность (анимации, туториал, дополнительные меню, экраны и т. д.) поощряется. 


Срок: 1-3 дня. Скорость выполнения задания имеет значение, однако не стоит забывать о качестве.
